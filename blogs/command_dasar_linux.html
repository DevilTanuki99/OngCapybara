Ini merupakan blog pertama yang saya tulis di web portofolio ini. Di kesempatan kali ini, saya akan membagikan command dasar yang paling sering digunakan di terminal linux.


🔧Navigasi dan Info Sistem

# pwd - Print Working Directory - Melihat dirimu ada di folder apa.

pwd

# ls - Lihat isi folder.

ls         # sederhana
ls -l      # detail (permissions, owner, size)
ls -lah    # human-readable + hidden files

# cd - masuk/move folder.

cd /path/to/folder
cd ~       # ke home
cd ..      # naik 1 level

# whoami - Nama user sekarang.

whoami

# hostname - nama mesin.

hostname

📁 File & folder (buat, lihat, hapus)

# mkdir - membuat folder.

mkdir new-folder
mkdir -p parent/child/grandchild  # buat parent bila belum ada

# touch - Membuat file kosong.

touch file.txt

# cat - Membaca file.

cat file.txt

# nano - Gui terminal untuk menulis file.

nano  # menulis file yang belum ada
nano file.txt  # membaca sekaligus mengedit file yang telah ada

# cp - Copy file/folder.

cp file.txt copy.txt
cp -r folder1 folder2   # copy folder rekursif

# mv - Pindah/rename.

mv oldname.txt newname.txt
mv file.txt /path/to/dest/

# rm - hapus/remove.

rm file.txt
rm -r folder_to_remove   # hapus folder rekursif
rm -i file.txt           # konfirmasi tiap file

🔐File Permission & Owner

# sudo - Superuser do ibaratkan Administrator di windows.

sudo su

# chmod - Mengubah permission.

chmod +x script.sh     # beri execute
chmod 755 script.sh     # numeric mode

# chown - Mengubah owner.

sudo chown user:group file.txt

Tips Supaya Mahir Pakai Terminal

  • Perbanyak latihan di terminal.
  • Tidak perlu mengingat semua command sekaligus, 1 pisau yang mahir lebih baik dari 100 pisau yang tak pandai kita gunakan.
  • Gunakan variasi command supaya lebih memudahkan kita dalam aktivitas di terminal.
  • Pelajari dasar sistem operasi Linux.
